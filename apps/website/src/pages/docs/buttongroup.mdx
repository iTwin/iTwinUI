Use a button group if you want to but similar functionality in one place. It helps users differentiate functional areas of the UI.

Buttons in button group can be active, disabled and both, active and disabled.

## Appearance

### Overflow

If there is not enough space to display all of the buttons, the buttons can truncate into an overflow menu.

## Usage

- The button group should only consist of default buttons or borderless buttons
- You should not mix different styles buttons in the same group.
- You can put buttons and button group in close proximity. In such cases, see if there is a main action and give it emphasis with a [high visibility](button#high-visibility) button.
