Input group class is a container to enhance an input by adding an icon, text, or a button in front or behind the input field as a "help text".

Multiple add-ons are supported and can be mixed with [checkbox](checkbox), [radio](radio), [toggle switch](toggleswitch) input version. To learn more about when to use a toggle switch or a checkbox, please read (https://uxplanet.org/checkbox-vs-toggle-switch-7fc6e83f10b8).
