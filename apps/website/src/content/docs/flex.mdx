## Usage

Wrap `Flex` around the components that should be the children of the flex container. Set the gap using the `gap` prop with a value of an iTwinUI size token (e.g. `x`, `xs`) or any valid css value (e.g. "1rem"). The default `gap` is `xs`.

<LiveExample src='Flex.main.jsx'>
  <AllExamples.FlexMainExample client:load />
</LiveExample>

The `Flex` container offers properties which reflect similar behaviors to the CSS flexbox, such as `alignItem`, `justifyContent`, `flexDirection`, and `flexWrap`.

### Spacer

The `Flex.Spacer` subcomponent fills up the available space between flex items. This subcomponent is useful when it is necessary to push certain items to the edges.

<LiveExample src='Flex.spacer.jsx'>
  <AllExamples.FlexSpacerExample client:load />
</LiveExample>

### Flex Items

This `Flex.Item` allows customizing flex items through the `flex`, `alignSelf` and `gapBefore`/`gapAfter` props.

<LiveExample src='Flex.flexItems.jsx'>
  <AllExamples.FlexWithFlexItemsExample client:load />
</LiveExample>

The `gapBefore`/`gapAfter` props can used to override the gap at an individual level, for when you need a different gap than the one set in the parent `Flex`.
