## Usage

By default, the `<Kbd>` component renders as a [`<kbd>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/kbd) HTML element. You can either pass custom key(s) as the `children` or use the `KbdKeys` enum which includes a convenient set of constants for common keyboard keys such as `KbdKeys.Command`, `KbdKeys.Enter`, and `KbdKeys.Right`.

```tsx
/* Custom KbdKeys  */
<Kbd>A</Kbd>

/* Predefined KbdKeys */
<Kbd>{KbdKeys.Enter}</Kbd>
```
