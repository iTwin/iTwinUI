## Usage

The `<Kbd>` component is a styled wrapper around the [`<kbd>` HTML element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/kbd). Here is what basic usage looks like:

You can either pass custom key(s) as the `children` or use the `KbdKeys` enum which includes some of the common keyboard keys:

- `KbdKeys.Command`
- `KbdKeys.Shift`
- `KbdKeys.Backspace`
- `KbdKeys.Enter`
- `KbdKeys.Eject`
- `KbdKeys.Control`
- `KbdKeys.Windows`
- `KbdKeys.Apple`
- `KbdKeys.Option`
- `KbdKeys.Left`
- `KbdKeys.Up`
- `KbdKeys.Right`
- `KbdKeys.Down`

```tsx
/* Custom KbdKeys  */
<Kbd>A</Kbd>
```

```tsx
/* Predefined KbdKeys */
<Kbd>{KbdKeys.Enter}</Kbd>
```
