## Usage

The `<Kbd>` component is a styled wrapper around the [`<kbd>` HTML element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/kbd).

### Basic

You can pass custom key(s) as the `children` parameter.

```tsx
<Kbd>A</Kbd>
```

### Predefined Key

You can also use the `KbdKeys` enum which includes a convenient set of constants for common keyboard keys.

```tsx
<Kbd>{KbdKeys.Enter}</Kbd>
```

List of supported keys:

- `KbdKeys.Command`
- `KbdKeys.Shift`
- `KbdKeys.Backspace`
- `KbdKeys.Enter`
- `KbdKeys.Eject`
- `KbdKeys.Control`
- `KbdKeys.Windows`
- `KbdKeys.Apple`
- `KbdKeys.Option`
- `KbdKeys.Left`
- `KbdKeys.Up`
- `KbdKeys.Right`
- `KbdKeys.Down`
