You can pass `StatusMessage` to any component that has `message` prop (e.g. form fields such as inputs, selects, etc.) to fully customize the rendered message.

```jsx
<LabeledInput
  placeholder='Enter text here...'
  label='This is a label'
  message={<StatusMessage startIcon={<SvgCamera />}>â¬… This is a custom icon</StatusMessage>}
/>
```

It can also be used as a child of [`InputGrid`](/input-grid) to display messages below the form fields.

```jsx
<InputGrid>
  <Label>This is label</Label>
  <Input />
  <StatusMessage>This is message</StatusMessage>
</InputGrid>
```

### Status

Use `status` prop to set proper color to the message.

### Icon

Use the `startIcon` prop to customize the icon.

- It will default to the `status` icon, if `status` is set. Else, it defaults to no icon.
- If `startIcon` is set to `null`, no icon will be displayed, even if `status` is set.
