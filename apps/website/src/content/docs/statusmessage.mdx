The `StatusMessage` can be used with form components that have a `message` prop (e.g. `LabeledInput`, `LabeledTextarea`, `LabeledSelect`, `ComboBox`, `InputGroup`)

```jsx
<LabeledInput
  label='This is the label'
  message={<StatusMessage>This is the message</StatusMessage>}
/>
```

It can also be used as a child of [`InputGrid`](/input-grid) to display messages below the form fields.

```jsx
<InputGrid>
  <Label>This is the label</Label>
  <Input />
  <StatusMessage>This is the message</StatusMessage>
</InputGrid>
```

### Status

Use `status` prop to set proper color to the message.

### Icon

By default, a status icon will be displayed if the `status` prop is set.

Use the `startIcon` prop to add a custom icon.

If `startIcon` is set to `null`, no icon will be displayed, even if `status` is set.
