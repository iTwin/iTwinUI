A toast notification provides a brief message about app processes at the top of the screen. Because these messages are so prominent, we need to be careful not to overuse or misuse them.

## Variants

### Status

A toast message can come with a status such as:

- `positive`
- `negative`
- `warning`
- `informational`

### Position changed

The toast position can be customized using the `placement` prop, which is default to `top`. The available `placement` values are:

- `bottom-end`
- `top`
- `top-start`
- `top-end`
- `bottom`
- `bottom-start`

`Start` indicates left side of viewport. `End` - right side of viewport.
