The Information Panel is designed for complex tables, offering a user-friendly way to explore content without overwhelming details or horizontal scrolling. It enhances data presentation in tables when needed.

## Usage

The Information Panel component is made up of customizable subcomponents. Hereâ€™s what the basic usage looks like:

```tsx
<InformationPanelWrapper>
  <Table ... /> {/* component to overlay */}

  <InformationPanel isOpen={true}>
    <InformationPanelHeader onClose={() => {}}>
      <Text variant='subheading'>InfoPanel heading</Text>
    </InformationPanelHeader>

    <InformationPanelBody>
      <Text>Metadata</Text>
      <InformationPanelContent ... />
      <Text>Attributes</Text>
      <InformationPanelContent ... />
    </InformationPanelBody>

  </InformationPanel>
</InformationPanelWrapper>
```

### Overlay Component

The `InformationPanel` component is designed to overlay content on the page. It is typically used to display additional information or actions without navigating away from the current view. By default, the panel is closed. It can be opened by setting `isOpen={true}`.

### InformationPanelWrapper

The `InformationPanelWrapper` is a container that is used to wrap and manage the placement of the `InformationPanel`. It helps in organizing and controlling the layout of the panel within the overall structure of the page.

### InformationPanelContent

The `InformationPanelContent` is used to display the content within `InformationPanel`. Inline/Block display style of the content can be set using the `displayStyle` prop.

### onClose

The `onClose` function is an event handler used to close or hide the `InformationPanel`. When this function is called, it sets the `openRowIndex` state to -1, effectively closing the panel.

## Horizontal Orientation

By default, the orientation of the Information Panel component is vertical. It can be changed by setting `orientation='horizontal'`.
