Input group is a container to enhance an input by adding an icon, text, or a button in front or behind the input field as a "help text".

## Usage

<LiveExample src='InputGroup.checkboxGroup.jsx'>
  <AllExamples.InputGroupCheckboxGroupExample client:load />
</LiveExample>

Multiple add-ons are supported and can be mixed with [checkbox](checkbox#with-input-group), [radio](radio#statuses), [toggle switch](toggleswitch#with-input-group) input version.
